# Site Review Prompt

Analisa o relatório gerado pelo RobotMCP e avalia se a aplicação web NetLearn Battle C# está suficientemente testada para entrega académica.

Verifica:

* se o build passou;
* se os testes xUnit passaram;
* se os testes Robot passaram;
* se os fluxos principais estão cobertos;
* se existem falhas;
* se faltam testes importantes;
* se há riscos antes da entrega.

Não sugerir grandes refatorações.
Não alterar código sem necessidade.
Priorizar estabilidade.

Responder com:

1. Estado geral.
2. Pontos fortes.
3. Riscos.
4. Melhorias recomendadas.
5. Se pode entregar.
